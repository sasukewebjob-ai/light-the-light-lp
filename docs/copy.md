## ① ターゲット

### 属性
- 20代〜60代
- 個人事業主／副業層／独立志向の会社員
- 職種：Webライター、ディレクター、広報PR、LP作成、新聞記者、デザイナー、整体師、CS等
- AIに興味はあるが実務・収益に繋げられていない方

### 悩み
- AIを触っているが成果に繋がらない
- 作業に時間がかかりすぎている
- 何から学べばいいか分からない
- 情報が多すぎて迷っている
- 一人だと継続できない
- 収益や事業運営に不安がある
- 経理・税務・経営の知識がない

## ② サービス内容

### コンセプト
AI活用を軸に「実務」「収益化」「事業運営」まで一気通貫でサポートする個人事業主・副業者向けコミュニティ。
単なる"学ぶ場"ではなく「事業者チーム」。

### 事業者チームの仕組み
- 個人で稼ぐだけでなく「チームとして仕事を受ける」
- コミュニティ内で仕事の受発注が発生
- 「学ぶ → 実践する → 収益化する」までをコミュニティ内で完結

## ③ ベネフィット

### ビフォーアフター
| Before | After |
|--------|-------|
| AIを触っているだけ | AIを「成果に直結する形」で使える |
| 作業に時間がかかる | 作業時間が大幅に短縮 |
| 迷い続ける | やるべきことが明確になる |
| 一人で継続できない | 継続できる環境がある |
| 孤独 | チームの中で仕事の機会が生まれる |

### 具体的な数字
- 10時間 → 3時間に短縮
- 資料作成時間が1/10に削減
- 作業時間が1/3に短縮
