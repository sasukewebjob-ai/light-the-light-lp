## 2026-05-02：5専門家AgentによるLP徹底議論＆改善実装

### 実施内容
DR/コピー/UX-CRO/ターゲット適合/グロースハッカーの5視点で並列評価。
全員一致の致命的問題（Heroキャッチ・CTA・登録後不透明）を中心に改修。

### バックアップ
- 実装直前の状態を `index-pre-ab.html` として保存（戻せるように）

### 実装した改善（保証は除外、ユーザー指示）

| 施策 | 内容 |
|---|---|
| **A** Heroキャッチ刷新 | 「AIを触るだけで終わっていませんか？」→「AIは学んだ。でも、仕事には繋がっていない。」（中級者にも刺さる射程拡大） |
| Hero CTA刷新 | メイン「今すぐ詳細を見る」→「LINEで詳細を受け取る（30秒）」（LINE直リンク）／サブ「料金プランを見る」 |
| **B** マイクロコピー | CTA直下に「※登録後、コミュニティの詳細案内が届きます。決済はありません。」追加 |
| **C** 固定フローティングCTA | スマホのみ表示、Hero通過後フェードイン、Final CTA表示中は非表示（IntersectionObserver制御） |
| **E** 比較アンカリング | Pricing冒頭に「個別AIコンサル相場 ¥15,000〜30,000 → 当サービス ¥3,500/回相当」ブロック |
| **J** 業種別ユースケース | Services直後に新セクション「こんな業種の方が成果を出しています」（ライター/広報/整体師/ディレクター） |
| **K** 入会後ロードマップ | Offer直前に4ステップ図解（Day1→入会後→Week1→Month1） |
| **L** FAQ追加 | 「申込後の流れ」「Slackについて」「料金は他社と比べて高くないか」を冒頭に追加 |
| **I** OGP/canonical | og:image・twitter:card・canonical等を`<head>`に追加（og:imageはseminar1.png流用） |
| **G** GA4プレースホルダー | 測定ID取得後に貼付できるよう、コメント形式でスクリプト枠を用意 |

### スキップした施策
- **D** リスクリバーサル（保証）：ユーザー指示で除外
- **H** UTMパラメータ：lin.ee短縮URLで保持されないため実装意義なし
- **F** お客様の声書換：虚偽リスクのためFAQで価格不安解消に振替
