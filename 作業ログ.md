作成日：2026-03-30

## 作業の流れ

### STEP 1：情報収集
- LP設計の理論（スライド画像）を読み込み
  - LPの構成要素の役割（選ばれる理由・共感・信頼・安心）
  - LPの鉄板構成（悩み解決型7ステップ）
  - 人が行動を起こすLPの5か条
  - ライフフォース・エイト（8つの根源的欲求）
- ターゲット・サービス内容・ベネフィット・実績・お客様の声・料金プラン・ストーリーをヒアリング

### STEP 2：CLAUDE.md作成
- `LP作成/CLAUDE.md` にプロジェクト情報を整理・保存

### STEP 3：index.html作成
- HTML1ファイルで完結するLP
- スマホ対応（レスポンシブ）
- Google Fonts（Noto Sans JP）使用
- スクロール連動フェードインアニメーション
- ゴールド×ネイビー×ホワイトのデザイン

### STEP 4：細部の修正（随時）
| 修正内容 | 詳細 |
|----------|------|
| お客様の声を4名→6名に | 40代女性Webディレクター・60代男性シナリオライターを追加、3×2グリッドに |
| プロフィール写真を追加 | `profile.jpg` を信頼セクションとストーリーセクションに配置 |
| 活動の様子セクション追加 | 勉強会画像4枚（seminar1〜4.png）をネイビー背景で表示 |
| 画像ファイル名の修正 | `seminar1.jpg.png` → `seminar1.png` にリネーム（bash） |
| プロフィール情報を更新 | 現役AIスクール講師200名以上・元会計事務所・YouTube約2万人 |
| メディアURLを追加 | 株式会社メイカヒット2件のURLをリンク付きで掲載 |
| CTAをグリーンに変更 | ゴールド→グリーン（#16A34A）に統一 |
| 料金プランセクションを追加 | コミュニティ強調カード＋個別Zoomプラン3種＋単発 |
| コミュニティの含まれる内容を修正 | AIコンサル等→月2回グループコンサル・仕訳会・Q&A等に |
| CTAにLINEリンクを設定 | https://lin.ee/Y7OfHPe |
| イニシャルを追加 | K・M・U・T・K・Kさん |
| （コミュニティなし）を赤字に | ベーシックプランに警告表示 |

### STEP 5：GitHub公開
1. `LP作成` フォルダで git init
2. `git remote add origin https://github.com/sasukewebjob-ai/light-the-light-lp.git`
3. `git add -A && git commit -m "Add ライトザライト LP" && git push -u origin main`
4. GitHub Pages を有効化（gh CLIで自動設定）
